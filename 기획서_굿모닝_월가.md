# 서비스 기획서: 굿모닝 월가 (Good morning, Wall Street)

## 1. 서비스 개요

"굿모닝 월가"는 한국인 미국주식 투자자를 위한 자동화된 데일리 브리핑 서비스입니다. 매일 한국 시간 새벽, 미국 증시가 마감된 후 가장 활발하게 거래된 종목들을 Yahoo Finance 데이터를 기반으로 자동 선정하고, 각 종목의 핵심 정보와 관련 뉴스를 수집하여 AI가 요약한 이미지 브리핑을 생성합니다. 사용자는 복잡한 분석이나 검색 없이 이메일 또는 슬랙을 통해 매일 아침 가장 중요한 미국주식 정보를 받아볼 수 있습니다. 특히 바쁜 직장인 투자자들이 출근 전 짧은 시간에 핵심 정보를 파악할 수 있도록 Chatless 방식으로 완전 자동화된 서비스를 제공합니다.

## 2. 핵심 가치 제안 (Value Proposition)

### "복잡한 분석 없이, 가장 활발한 종목만 자동으로"

- **시간 절약**: 매일 수십 개의 종목을 직접 분석하고 뉴스를 찾아볼 필요 없이, 가장 중요한 정보만 선별하여 제공
- **객관적 선정**: 거래량과 상승률 데이터 기반으로 감정이나 주관적 판단 없이 시장에서 실제로 가장 주목받는 종목 자동 선정
- **완전 자동화**: 사용자 입력 없이 매일 정해진 시간에 자동으로 브리핑 생성 및 발송 (Chatless)
- **한국인 맞춤**: 한국어로 제공되는 정보와 한국 투자자 관점에서 이해하기 쉬운 형식의 브리핑
- **주식 초보자 맞춤형**: 현재가, 거래량, 시가총액, P/E 비율 등 주요 지표를 쉬운 말과 예시로 해설하여 전문 지식 없이도 정보를 이해할 수 있도록 지원
- **빠른 의사결정**: 출근 전 짧은 시간에 핵심 정보만 파악하여 투자 결정에 필요한 인사이트 제공

## 3. 주요 기능 목록

1. **Yahoo Finance 기반 화제 종목 자동 선정**
   - 전일 거래량과 상승률 데이터 수집 및 분석
   - 상위 N개 종목 자동 선정 알고리즘

2. **종목 정보 자동 수집 및 초보자용 해설**
   - 선정된 종목의 기본 정보 (현재가, 전일 대비, 거래량 등)
   - 시가총액, P/E 비율 등 주요 지표
   - 각 지표에 대한 쉬운 해설과 예시 제공 (전문 용어 없이 누구나 이해 가능)

3. **관련 뉴스 자동 수집 및 요약**
   - 각 종목 관련 최신 뉴스 수집
   - AI를 활용한 뉴스 핵심 내용 요약

4. **AI 이미지 브리핑 자동 생성**
   - 종목 정보와 뉴스 요약을 시각적으로 정리한 이미지 브리핑 생성
   - 모바일/데스크톱 최적화된 레이아웃

5. **이메일 자동 발송**
   - 매일 정해진 시간에 브리핑 이미지를 이메일로 자동 발송
   - 사용자 맞춤형 발송 시간 설정 가능

6. **슬랙 자동 발송**
   - 슬랙 웹훅을 통한 자동 발송 기능
   - 팀 단위 정보 공유 지원

7. **발송 이력 및 통계 제공**
   - 과거 브리핑 이력 조회
   - 선정된 종목의 후속 성과 추적 (선택적)

## 4. 사용자 시나리오 (User Journey)

### 시나리오 1: 신규 사용자 등록 및 첫 브리핑 수신

1. **인지 (Awareness)**
   - 미국주식 투자 커뮤니티나 SNS에서 "굿모닝 월가" 서비스 소개글 발견
   - "매일 자동으로 화제 종목만 받아볼 수 있다"는 점에 관심

2. **등록 (Registration)**
   - 웹사이트 또는 간단한 랜딩 페이지에서 이메일 주소 입력
   - 발송 시간 선택 (기본값: 오전 7시)
   - 슬랙 연동 선택 (선택사항)

3. **첫 경험 (First Experience)**
   - 다음날 오전 7시, 이메일로 첫 브리핑 수신
   - 이미지 형태의 깔끔한 브리핑 확인
   - 전일 가장 활발했던 5개 종목과 핵심 정보 한눈에 파악

4. **일상화 (Habit Formation)**
   - 매일 아침 출근 전 브리핑 확인이 루틴화
   - 브리핑을 통해 파악한 종목으로 투자 결정에 활용

### 시나리오 2: 기존 사용자의 일상적인 사용

1. **자동 수신 (Daily Reception)**
   - 매일 오전 7시, 이메일/슬랙으로 브리핑 자동 수신
   - 별도 앱 실행이나 웹사이트 방문 불필요

2. **빠른 정보 파악 (Quick Scan)**
   - 출근 준비 중 또는 지하철에서 모바일로 브리핑 확인
   - 이미지 형태로 제공되어 2-3분 내 핵심 정보 파악

3. **투자 결정 지원 (Decision Support)**
   - 브리핑에서 관심 있는 종목 발견 시, 추가 조사 진행
   - 브리핑의 뉴스 요약을 바탕으로 투자 판단

4. **피드백 및 개선 (Feedback)**
   - 특정 종목이 계속 선정되는 경우, 해당 종목에 대한 추가 정보 요청 (향후 기능)
   - 발송 시간 조정 등 개인화 설정 변경

## 5. 성공 지표 (KPI)

| **사용자 만족도 (NPS)** | 서비스 추천 의향 점수 | 50점 이상 |
