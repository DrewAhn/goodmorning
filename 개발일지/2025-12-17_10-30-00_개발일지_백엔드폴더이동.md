## 해결하고자 한 문제
- FastAPI 백엔드 폴더(`backend`)가 `goodmorning` 폴더 바깥(프로젝트 루트)에 생성되어 있었음
- Git 저장소에서 프론트엔드와 백엔드를 함께 관리하기 위해 폴더 구조 정리 필요

## 해결된 것
- `backend` 폴더를 `goodmorning/backend`로 이동 완료
- 현재 폴더 구조:
  ```
  D:\ICA\1주차\
  ├── goodmorning\
  │   ├── backend\          ← 백엔드 (FastAPI)
  │   │   ├── api\
  │   │   ├── models\
  │   │   ├── services\
  │   │   ├── main.py
  │   │   ├── requirements.txt
  │   │   └── venv\
  │   ├── src\              ← 프론트엔드 (Next.js)
  │   └── ...
  └── ...
  ```

## 해결되지 않은 것
- 특별히 없음

## 향후 개발을 위한 컨텍스트
- 백엔드 서버 실행 시 경로가 변경되었으므로 주의 필요
  - 이전: `D:\ICA\1주차\backend`
  - 현재: `D:\ICA\1주차\goodmorning\backend`
- 가상환경 활성화 경로:
  ```powershell
  cd D:\ICA\1주차\goodmorning\backend
  .\venv\Scripts\Activate.ps1
  ```
- 서버 실행:
  ```powershell
  uvicorn main:app --reload
  ```
- 프론트엔드와 백엔드가 같은 goodmorning 폴더 내에 있어 한번에 Git 커밋 가능
