# 굿모닝 월가 - AGENT 모드 프롬프트 모음

> Cursor AI AGENT 모드에서 사용한 프롬프트 원문 정리

---

## 확장기능 1: 주가 차트 추가

```
현재 대시보드에 화제 종목 별 주가 차트를 추가해주세요.

[요구사항]
1. 기본로드 : 최근 5일 차트만, 선택 옵션 탭 5일/1개월/3개월
2. 라인 차트
2. 거래량 바 차트 (하단)
3. 차트 라이브러리: recharts 또는 chart.js
4. 목업 데이터 사용
5. 상승/하락에 따른 색상 변경 (녹색/빨간색)
6. 기간 별 한줄 해설 (규칙 기반)

[위치]
StockCard 컴포넌트, 지표 섹션과 초보자 해설 섹션 사이

[데이터 로드 방식]
사용자가 옵션 탭(5일/1개월/3개월) 클릭 시 로드 + 캐싱, 다시 5일로 돌아와도 클릭 및 로딩이 일어난 1개월, 3개월 데이터 유지
```

---

## 확장기능 2: 다크/라이트 모드 토글

```
다크 모드와 라이트 모드를 전환할 수 있는 기능을 추가해주세요.

[요구사항]
1. 헤더에 토글 스위치 배치
2. 아이콘: 🌙 (다크) / ☀️ (라이트)
3. 시스템 설정 감지 (prefers-color-scheme)
4. 로컬스토리지에 선택값 저장
5. 전환 시 부드러운 애니메이션
```
